# TrustEvo2D ─ 二維空間信任度與認知演化模擬器

**TrustEvo2D** 是一個結合了「二維空間物理碰撞體系」、「連續型重複囚徒困境賽局」與「認知過濾偏誤演化」的高保真度智能體（Agent）生態演化模擬實驗沙盒。

🔗 **線上演示 (Live Demo)**: [https://toydogcat.github.io/sim-trust-evolution/](https://toydogcat.github.io/sim-trust-evolution/)

## 🔬 空間賽局與經濟收益計算公式

當沙盒中的兩個小球（智能體 $A$ 與 $B$）發生二維物理碰撞時，會立即觸發一場**連續型囚徒困境賽局（Continuous Prisoner's Dilemma）**，其經濟收益與能量更迭完全遵循以下數學模型：

### 1. 對手期望合作度計算 ($E$)
每個小球大腦中都有一個交往記憶庫，記錄了與每位對手交往的歷史合作出價紀錄。
小球會根據與該對手的交往史計算其「期望合作度」 $E \in [0, 1]$：
$$E = \begin{cases} 
\frac{\sum_{i=1}^{n} P_{\text{mem}, i}}{n} & \text{若有歷史交往紀錄 } (n > 0) \\
0.5 & \text{若無交往歷史 (陌生碰面)}
\end{cases}$$
*其中 $P_{\text{mem}, i}$ 為記憶庫中記錄的對手合作出價，$n$ 為記憶深度（受大腦記憶容量 $M$ 限制）。*

### 2. 行為決策出價 ($P$)
小球根據其遺傳的**行為決策風格（Behavioral Style）**，對該期望值進行反應，輸出其本次合作出價 $P \in [0, 1]$（$P = 1$ 代表完全合作，$P = 0$ 代表完全背叛）：

*   **以牙還牙交易型 (Trader)**：對等回報，期望對方如何對待自己，自己就如何回報。
    $$P = E$$
*   **加倍懷疑自保型 (Defensive)**：保守防衛，對合作持懷疑態度，出價呈平方遞減。
    $$P = E^2$$
*   **欺善怕惡割韭菜型 (Exploiter)**：反向壓榨，對方越值得信任時，越傾向出低價背叛以割取最大利益。
    $$P = 1.0 - E$$
*   **大愛破冰聖人型 (Saint)**：主動破冰，積極釋放善意，即使預期不高也願意主動信任，出價呈根號攀升。
    $$P = \sqrt{E}$$

### 3. 碰撞收益與能量增幅公式 ($\Delta W$)
設小球 $A$ 的出價為 $P_A$，小球 $B$ 的出價為 $P_B$。碰撞後雙方的能量（財富）增幅 $\Delta W$ 計算公式如下：
$$\Delta W_A = k \times [ P_B \times (2 - P_A) - P_A ]$$
$$\Delta W_B = k \times [ P_A \times (2 - P_B) - P_B ]$$

*其中 $k$ 為環境收益協同乘數（系統參數 `payoffMultiplier`，預設為 $10$；若開啟「物資緊縮」模式，則 $k$ 折半為 $5$）。*

#### 💡 收益情境解析
這套公式完美對應了傳統「囚徒困境」的博弈矩陣，並將其平滑至 $[0, 1]$ 連續空間：
*   **雙方完全合作 ($P_A=1, P_B=1$)**：
    $$\Delta W_A = k \times [ 1 \times (2 - 1) - 1 ] = 0$$
    *（雙方和諧共處，無額外利益剝削與重大摩擦損耗，但需持續扣除每步生存成本）*
*   **一方完全背叛，一方完全合作 ($P_A=0, P_B=1$)**：
    $$\Delta W_A = k \times [ 1 \times (2 - 0) - 0 ] = +2k \quad \text{(A 成功收割，獲最大暴利)}$$
    $$\Delta W_B = k \times [ 0 \times (2 - 1) - 1 ] = -1k \quad \text{(B 盲目信任，承受最高損失)}$$
*   **雙方完全背叛 ($P_A=0, P_B=0$)**：
    $$\Delta W_A = \Delta W_B = 0$$
    *（雙方無交易發生，不產生多餘損耗，但依然面臨每步生存成本扣發，陷入雙輸破產危機）*

---

### 4. 認知偏誤記憶過濾器 (Memory Filter)
賽局完成後，小球在把對手的真實出價 $P_{\text{real}}$ 存入記憶時，會經過其遺傳的**認知過濾器（Memory Filter）**修飾，形成主觀認知偏差：

*   **客觀實記派 (Objective)**：實事求是，如實記錄對手的出價。
    $$P_{\text{mem}} = P_{\text{real}}$$
*   **樂觀記善美化派 (Optimist)**：只記好不記壞。
    *   若本輪交易獲利（$\Delta W \ge 0$），記錄真實出價：$P_{\text{mem}} = P_{\text{real}}$
    *   若本輪遭受損失（$\Delta W < 0$），心靈過濾，記錄美化後的數據：$P_{\text{mem}} = \min(P_{\text{real}} + 0.3, 1.0)$
*   **悲觀疑嫌記仇派 (Pessimist)**：擴大敵意。
    *   若本輪遭受損失（$\Delta W < 0$），記住痛苦，如實記錄：$P_{\text{mem}} = P_{\text{real}}$
    *   若本輪交易获利（$\Delta W \ge 0$），保持狐疑，記錄打折後的數據：$P_{\text{mem}} = \max(P_{\text{real}} - 0.2, 0.0)$

## 🧬 生態自然選擇與繁衍突變

1.  **生存成本 (Survival Cost)**：每一步 tick 所有小球皆會被扣除微量生存能量（由滑桿控制，如 `0.20 / tick`）。當小球財富（能量）降至 $\le 0$ 時，宣告死亡並從地圖中溶解。
2.  **繁衍裂變 (Fission)**：當小球累積的能量超過繁衍閾值（常規為 $200$ 金幣，地獄模式為 $350$ 金幣）時，小球會扣除一半能量，原處分裂出一個繼承大腦與歷史記憶的子代。
3.  **遺傳突變 (Mutation)**：子代繁衍時有一定機率（突變率）發生基因重組突變，隨機改變子代的**認知過濾器類型**、**行為決策風格**或**記憶容量**。
